# 🚀 Git & GitHub Learning Journey

## 📖 Overview

This repository contains my hands-on learning and practice of Git & GitHub concepts, including version control, branching strategies, collaboration workflows, and advanced Git commands.

---

## 🛠️ Core Concepts

| Concept | Description |
|----------|------------|
| Git | Version Control System used to track code changes |
| GitHub | Remote platform for storing and collaborating on Git repositories |
| Repository | Storage location for project files and version history |
| Branch | Independent line of development |
| Commit | Snapshot of changes in the repository |
| Merge | Combines changes from different branches |
| Remote | External Git repository hosted on GitHub |

---

## 📚 Commands Learned

<details>
<summary>1️⃣ Repository Setup</summary>

| Command | Purpose |
|----------|---------|
| `git init` | Initialize a new Git repository |
| `git clone <url>` | Clone a repository |
| `git remote add origin <url>` | Connect local repo to GitHub |
| `git remote -v` | View configured remotes |

</details>

<details>
<summary>2️⃣ Staging & Commit</summary>

| Command | Purpose |
|----------|---------|
| `git add .` | Stage all changes |
| `git add <file>` | Stage a specific file |
| `git commit -m "message"` | Create a commit |
| `git status` | Check repository status |
| `git diff` | View changes before commit |

</details>

<details>
<summary>3️⃣ Branch Management</summary>

| Command | Purpose |
|----------|---------|
| `git branch` | List branches |
| `git branch -a` | Show local & remote branches |
| `git checkout -b feature/name` | Create and switch branch |
| `git switch branch-name` | Switch branches |
| `git branch -m old new` | Rename branch |
| `git branch -d branch-name` | Delete local branch |

</details>

<details>
<summary>4️⃣ Remote Operations</summary>

| Command | Purpose |
|----------|---------|
| `git push origin main` | Push code to GitHub |
| `git push -u origin branch-name` | Set upstream branch |
| `git pull origin main` | Pull latest changes |
| `git fetch origin` | Fetch remote changes |
| `git pull --rebase origin main` | Rebase while pulling |

</details>

<details>
<summary>5️⃣ Merge & Conflict Resolution</summary>

| Command | Purpose |
|----------|---------|
| `git merge feature-name` | Merge branch |
| `git merge --abort` | Cancel merge operation |
| `git log --graph --oneline --all` | Visualize branch history |

</details>

<details>
<summary>6️⃣ Stash Operations</summary>

| Command | Purpose |
|----------|---------|
| `git stash` | Temporarily save changes |
| `git stash list` | Show stashes |
| `git stash apply` | Restore stash |
| `git stash pop` | Restore and remove stash |

</details>

<details>
<summary>7️⃣ Advanced Commands</summary>

| Command | Purpose |
|----------|---------|
| `git reset` | Unstage changes |
| `git reset --soft HEAD~1` | Undo commit, keep staged |
| `git reset --hard HEAD~1` | Remove commit permanently |
| `git commit --amend` | Modify latest commit |
| `git revert <commit>` | Revert a commit safely |
| `git cherry-pick <commit>` | Copy specific commit |
| `git show <commit>` | View commit details |
| `git bisect` | Find bug-causing commit |

</details>
