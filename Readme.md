# Linux Command Line Practice

This repository contains my hands-on Linux command line practice and notes. It covers essential Linux administration, file management, process management, networking, package management, and system monitoring commands commonly used in Linux environments.

## File & Directory Management

| Command | Description                          |
| ------- | ------------------------------------ |
| `pwd`   | Display current working directory    |
| `ls`    | List files and directories           |
| `cd`    | Change directory                     |
| `mkdir` | Create a new directory               |
| `rmdir` | Remove an empty directory            |
| `rm`    | Delete files and directories         |
| `touch` | Create an empty file                 |
| `cp`    | Copy files and directories           |
| `mv`    | Move or rename files and directories |

## File Viewing & Search

| Command | Description                     |
| ------- | ------------------------------- |
| `cat`   | Display file contents           |
| `less`  | View file contents page by page |
| `head`  | Display first lines of a file   |
| `tail`  | Display last lines of a file    |
| `grep`  | Search text patterns in files   |
| `find`  | Search files and directories    |

## Process Management

| Command | Description                           |
| ------- | ------------------------------------- |
| `ps`    | Display running processes             |
| `top`   | Monitor system processes in real time |
| `kill`  | Terminate a process using PID         |

## User & Permission Management

| Command   | Description                               |
| --------- | ----------------------------------------- |
| `chmod`   | Change file permissions                   |
| `chown`   | Change file ownership                     |
| `adduser` | Create a new user account                 |
| `passwd`  | Change user password                      |
| `usermod` | Modify user account settings              |
| `su`      | Switch user account                       |
| `sudo`    | Execute commands with elevated privileges |
| `whoami`  | Display current logged-in user            |

## Archiving & Compression

| Command | Description                      |
| ------- | -------------------------------- |
| `tar`   | Create and extract archives      |
| `zip`   | Compress files into ZIP archives |
| `unzip` | Extract ZIP archives             |

## Memory & Disk Management

| Command | Description                        |
| ------- | ---------------------------------- |
| `free`  | Display memory usage statistics    |
| `df`    | Show filesystem disk space usage   |
| `du`    | Show directory and file disk usage |

## Networking

| Command    | Description                             |
| ---------- | --------------------------------------- |
| `ping`     | Test network connectivity               |
| `ssh`      | Securely connect to remote servers      |
| `scp`      | Securely transfer files between systems |
| `curl`     | Transfer data using URLs and APIs       |
| `wget`     | Download files from the internet        |
| `ifconfig` | Display network interface information   |
| `ip`       | Configure and display IP settings       |
| `route`    | View and manage routing tables          |
| `netstat`  | Display network connections and ports   |
| `iptables` | Manage firewall rules                   |

## Package Management

| Command | Description                                |
| ------- | ------------------------------------------ |
| `apt`   | Package manager for Ubuntu/Debian systems  |
| `yum`   | Package manager for Red Hat/CentOS systems |

## System Administration

| Command      | Description                     |
| ------------ | ------------------------------- |
| `systemctl`  | Manage system services          |
| `journalctl` | View system logs                |
| `hostname`   | Display or set system hostname  |
| `uname`      | Display system information      |
| `date`       | Display current date and time   |
| `echo`       | Print text to terminal or files |
